# QR Generator Pro

Proyecto base en HTML/CSS/JS.

## Funciones incluidas
- Generación de QR en tiempo real.
- Cambio de color del QR.
- Cambio de color de fondo.
- Inserción de logo.
- Descarga en PNG.

## Próximas funciones
- Marca de agua.
- Gradientes.
- Formas personalizadas.
- Exportación SVG/JPG.
- Plantillas.
